Electrokinetic Injection (EKI) COMSOL Simulation

Multiphysics EC + Stokes + Nernst–Planck · pinch & dispense in Micronit X8050 · plug metrics (center/FWHM/velocity) for experiment matching

Objective

Build a simulation framework that reproduces the EKI pinch–dispense sequence in a cross-channel chip, so that experimental plug motion and broadening can be used to infer unknown parameters (e.g., zeta potential, effective diffusivity, unintended pressure offsets).

Outcome

  • Depth-averaged 2D Micronit X8050 cross-channel model with electroosmotic slip at walls
  • Pinching + dispensing voltage configurations with stationary fields followed by time-dependent transport
  • Quantitative plug metrics extracted downstream: center position, FWHM evolution, and plug velocity

Highlights

  • Coupled Electric Currents, Creeping Flow (Stokes), and Transport of Diluted Species
  • Mesh convergence focused on the junction + first downstream segment to stabilize plug tracking
  • Clear separation of mechanisms: EOF + electrophoresis + convection set velocity; diffusion + field stretching set broadening

Tools & Skills

COMSOL Multiphysics · Microfluidics · EOF/Electrophoresis · Nernst–Planck Transport · Mesh Convergence · Data Extraction
