Jumper Robot

36″ vertical jump via optimized spring-motor actuation · 2nd Place (UCSB ME153)

Objective

Design the highest-jumping robot using limited materials and rapid iterations.

Outcome

  • 36″ vertical jump (3× improvement over prototype)
  • Reliable force-controlled latch; low-friction string path
  • Momentum/energy modeling and high-speed testing

Highlights

  • Mass distribution optimization for takeoff velocity
  • Iterative tuning of spring tension and latch geometry
  • MATLAB plots for performance analysis

Tools & Skills

SolidWorks · Dynamics · Prototyping · MATLAB · Motor Control
