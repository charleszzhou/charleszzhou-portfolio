Torsional Stiffness Measurement for Soft Robotics

Low-Cost Experimental Rig for Quantifying Torsional Compliance

Executive Summary

Designed and built a low-cost experimental apparatus to measure the torsional stiffness of soft robotic actuators. The rig achieves ~0.1 Nm/deg resolution with <±5 % trial variance, enabling repeatable stiffness characterization without the need for an expensive commercial torque sensor (~$12k).

Problem & Purpose

Torsional compliance is a key property in soft robotic structures, but is difficult to measure accurately due to nonlinear deformation and the high cost of precision torque sensors. This project addresses the need for a repeatable, low-cost method to quantify torque–angle relationships across different soft actuator designs and pressurization states.

System Overview

The apparatus converts applied loads into known torques using a pulley-based moment arm, while angular deflection is measured via a bearing-mounted protractor assembly. Modular, swappable couplers allow the same setup to be used across different actuator geometries and sizes. A consistent calibration procedure ensures reproducible torque and angle measurements.

Key Outcomes

  • High resolution & repeatability: Achieved ~0.1 Nm/deg resolution with <±5 % variance across repeated trials.
  • Modular and adaptable: Interchangeable couplers support multiple actuator geometries and loading conditions.
  • Validated stiffness trends: Measured consistent torsional stiffness–pressure relationships in inflatable soft beams, enabling quantitative comparison between designs.

Design Highlights

  • Fully 3D-printed, modular mechanical structure
  • Repeatable calibration workflow for torque and angle measurement
  • Data analysis pipeline for torque–angle regression and stiffness extraction
