CubeMars Motor Control GUI

Python GUI for CAN-based MIT-mode control · live tuning & logging · safety watchdog

Objective

Develop a Python interface to control CubeMars AK-series BLDCs over CAN using MIT mode.

Outcome

  • Real-time multi-motor control with torque/position feedback
  • Heartbeat watchdog, soft torque limits, E-stop
  • Data logging and parameter presets for research testing

Highlights

  • Tkinter GUI + python-can (Seeed USB-CAN)
  • Daisy-linked CAN H/L; synchronized commands
  • Adaptive PID with live tuning sliders

Tools & Skills

Python · Tkinter · CAN Protocol · Embedded Systems · Data Visualization
